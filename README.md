# 雅思课程产品匹配器

一个无需构建的静态网页工具，用于根据学生目前雅思分数、目标分数、线上/线下偏好、课程形态偏好和备考周期，推荐匹配的班课、一对一或组合课程。

## 功能

- 输入目前雅思总分与目标分数。
- 选择线上、线下或混合授课。
- 选择班课、一对一、组合课或自动匹配。
- 展示匹配度最高的 3 个课程方案。
- 每个方案包含课程优势、适合情况、上课方式、周期和价格。

## 本地预览

直接打开 `index.html`，或在项目目录运行：

```bash
python3 -m http.server 4173
```

然后访问 <http://localhost:4173>。

## 产品数据维护

课程数据集中维护在 `script.js` 的 `courses` 数组中。后续如果有更准确的产品名称、价格、课时或卖点资料，可直接替换对应字段。
