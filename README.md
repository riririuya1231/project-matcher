# 雅思课程咨询方案生成器

一个无需构建的静态网页工具，用于咨询时快速生成雅思课程推荐方案。工具不再按备考时间匹配，而是根据学生目前雅思总分、目标总分、可选填小分、授课偏好和课程偏好，判断应该补单项还是全科，并输出可直接讲给学生/家长的课程方案。

## 功能

- 输入目前雅思总分与目标雅思总分，自动计算总分差距。
- 小分选填：可填写听力、阅读、口语、写作；未填写时默认先给全科体系课方案。
- 根据小分短板判断补单项、补输出项/输入项，还是全科滚动补弱项。
- 推荐一对一加班课小时数、约几次课、预估最低价，以及加班课应该怎么上。
- 展示班课、线上 1V1、线下全能/连报、飞跃计划等产品方案。
- 每个方案包含官方价、最低/会战价、课时、有效期、CP 码、服务优势与咨询话术。
- 页面顶部提供咨询前可铺垫的机构优势、雅思分数常见用途和推荐逻辑。

## 本地预览

直接打开 `index.html`，或在项目目录运行：

```bash
python3 -m http.server 4173
```

然后访问 <http://localhost:4173>。

## 产品数据维护

课程数据集中维护在 `script.js` 的 `products` 数组中。后续如果官方价、最低价、课时、CP 码、服务内容或重读条件有更新，可直接替换对应字段。
